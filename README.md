# TikTok Exploratory Data Analysis

## Project Overview

This project was completed as part of the Google Data Analytics Professional Certificate.

The objective was to explore a TikTok dataset using Python to understand its structure, summarize key variables, identify data quality issues, and generate insights.

## Tools Used

- Python
- pandas
- NumPy
- Jupyter Notebook

## Skills Demonstrated

- Data Import
- Data Inspection
- Data Cleaning
- Descriptive Statistics
- Exploratory Data Analysis (EDA)

## Files

- project.ipynb
- tiktok_dataset.csv
- README.md

## Key Findings

- Explored dataset structure
- Identified missing values
- Examined variable distributions
- Generated summary statistics
